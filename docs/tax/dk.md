# DK Tax — Notes (placeholder)

This page will document the Danish tax rules used by the tax engine, including:

- Lagerbeskatning (mark-to-market) for ETFs on the ABIS list
- Realisation taxation for non-ABIS instruments (gennemsnitsmetoden)
- Aktiesparekonto (ASK) — separate 17 % regime
- PAL-skat on PFA pension returns

Authoritative source: [skat.dk](https://skat.dk). Interpretations applied in code are recorded as ADRs.
